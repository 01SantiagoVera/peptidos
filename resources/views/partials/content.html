About Us

Our platform is a Python 3 web application designed to predict antimicrobial peptides (AMPs) using advanced machine learning techniques, including Random Forest, Neural Networks, and Support Vector Machines. The application leverages comprehensive protein sequence descriptors, such as Composition, Transition, and Distribution (CTD), to analyze peptide sequences.


With improved portability and accuracy compared to similar tools, our system offers researchers the ability to process peptide data, generate meaningful predictions, and optimize models using novel training datasets. Antimicrobial Predictor provides an intuitive interface, robust data management, and downloadable reports, empowering innovation in biomedical research and antimicrobial drug development.
